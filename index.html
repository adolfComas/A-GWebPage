Protección Contra Incendios - Sistemas de Detección y Alarmas

  • Diseño, suministro, instalación y mantenimiento de sistemas de detección de incendios bajo normas y está NFPA.
  • Tegnologia avanzada para un aviso oportuno.
  • Sistemas de audio evacuación integrado, que permiten una mejor información y mayor agilización de evacuaciones.
  • Sistemas de tipo direccionables, convencional, detección y alerta temprana mediante aspiración y del tipo de temperatura lineal (protectowire).

Protección Contra Incendios - Sistemas de Extinción de Incendios

  • Diseño, suministro, instalación y mantenimiento de sistemas de extinción de incendios.
  • Sistemas salas de bomba contra incendios.
  • Sistemas de rociadores automáticos.
  • Hidrantes, monitores automáticos, grifos y siamesas.
  • Sistemas de protección para cocinas, campanas, ductos de extranción, entre otros.
  • Sistemas de agente límpio(FM-200, Novec1230, gases inertes) y sistemas de CO2 de alta presión, para la pretección de equipos críticos, informáticos o alto valor.

Protección Contra Incendios - Sistemas de Extinción de incendios base a Agua

  • Diseño, suministro, instalación y mantenimiento de sistemas de extinción de incendios.
  • Sistemas salas de bomba contra incendios.
  • Sistema de rociadores automáticos.
  • Sistemas de agua pulverizada.
  • Hidrantes, monitores automáticos, grifos y samesas
  • Sistemas de gabinetes de mangeras rígidas, colapsables e industriales
  • Sistemas de extinción en base a espuma

Seguridad Electrónica - CCTV

  • Visualizació en tiempo real en áreas de operación, almacenaje, insfraestructuras, entre otros.
  • Grabación por perídodos definidos, indefinidos, movimientos o analísticas especificadas por el cliente para revisar la actividad de un sector determinado.
  • Permite monitoreo remoto desde teléfonos móviles, tablet o computador personal a través de internet.
  • Sistemas análogos, IP e inalámbricos.
  • Integración con sistemas remotos, control de acceso y/o control centralizado BMS.

Seguridad Electrónica - Sistema de Control de Acceso

  • Control de acceso a zonas restringidas.
  • Control de acceso vehicular mediante sistema Tag, lectura de PPU, proximidad, entre otros.
  • Soluciones integrables a control centralizado, BMS y CCTV.
  • Diversidad de soluciones adaptables a cada requerimiento ya sea de proximidad, biométrico, torniquetes, contraseña, esclusas, entre otras.

Seguridad Electrónica - Sistema de alarma e instrusión

  • Estos sistemas permiten detectar la presencia de intrusos mediante una gran variedad de productos (sensores de puerta y ventanas, detector de sonidos, sensor de movimiento, entre muchos otros) de acuerdo a la necesidad en particular.
  • Notificación en tiempo real vía app, llamada, mensaje de texto o correo electrónico.
  • Solución integrable con sistemas de control de acceso, BMS y CCTV.
  • Gran variedad de soluciones mediante tecnologías de bus e inalámbricas según sea la necesidad.
